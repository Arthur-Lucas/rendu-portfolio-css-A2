Site Web Fnatic

Mon premier Site Web,
Je l'ai créé lors de mon année préparatoire à l'IIM, soit ma premiere année (2020-2021).
Pour le projet de fin d'année nous avions pour projet de développer un site sur un thème choisi.
J'ai donc décidé de sélectionner l'équipe d'E-sport Fnatic, principalement connu de par sa qualité de jeu sur le jeu League of Legends
Le site devait contenir, une Home page, une page listant les joueurs de l'équipe, l'histoire de cette derniere, ainsi qu'une page gallery, contact et une page de connexion utilisateur/admin.
Pour le dévellopement de mon site, j'ai dû utiliser les langages HTML/CSS pour la réalisation du front-end du site; JS pour les animations, différentes fonctionnalités ainsi que pour la conception d'un dark-mode. Enfin j'ai dû utiliser le PHP/SQL pour lier ma base de données et permettre un systeme de connexion, administration ainsi qu'un systeme de commentaire.
